## What does it do?

It draws pretty graphics such as.

```{r, echo=FALSE}
library(ggplot2)
climate <- read.csv("./data/climate.csv")

x <- climate[which(climate$Country %in% "Africa"),]
   
g <- ggplot(data = x, aes(x = year, y = YearlyMean)) +
      geom_point() +
      geom_smooth(method = lm)
```
